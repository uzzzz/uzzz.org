      西瓜和洋葱的故事是这样开始的。洋葱本来就是西瓜身边的一只大尾巴狼，表面上和善可亲，事实上心里总是打着自己的小算盘。恋爱这种事情，男生总是主动些，所以有一天大尾巴狼找了一个借口和西瓜吃饭。这一顿饭，最终成为了一场鸿门宴。饭局里，洋葱问西瓜：“你觉得我追你怎么样？”也许是因为觉得这一问题非常没有层次，问法也不够讲究，没有人愿意回答这个问题。当时的气氛很尴尬，没人知道说什么好，饭没吃完（每次和西瓜吃正餐都是要剩下东西的），两人就匆匆离开了。故事到这里似乎就要结束了，但是没有波折怎么叫…
