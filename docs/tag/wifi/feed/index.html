]]> 输入WIFI密码，连接按钮显示为灰色 https://uzzz.org/article/3216.html Mon, 21 Jan 2013 06:23:02 +0000 https://uzzz.org/article/3216.html

问题描述：

在手机连接WIFI的时候，输入密码后“连接”仍是灰色显示，只有当输入>=8 位的时候“连接”按钮才能正常按下。

手机系统为:android 4.0

wifi密码是：7位


原因找到了，就像上文提到的，WIFI密码必须>=8位。

可参考一下：

http://en.wikipedia.org/wiki/Wi-Fi_Protected_Access#Security_.26_Insecurity_in_pre-shared_key_mode
