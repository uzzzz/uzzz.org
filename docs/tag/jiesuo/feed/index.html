手机锁屏忘记密码，可以通过adb进行解屏安全不丢失东西

1)下载adb.zip,并解压到D盘

2)通过dos进入d盘的adb目录下，cmd  输入D:\adb”并回车，然后输入”D:”回车;

3)输入”adb shell”并回车

4)输入”su”并回车

5)输入”cd data/system”

6)输入”ls”查看system目录下的文件

7)看到有一个”password.key”,输入“rm password.key”删除此文件;

8)重启手机就可以了

当然了还可以通过刷机、或双清来解决，但是我觉得通过adb才是安全的

 

adb工具下载地址：http://download.csdn.net/detail/niuba123456/9088571
