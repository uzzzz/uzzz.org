前言

  又有一段时间没有更新了。一大清早就被一则A站被脱库的新闻所吸引。事件的影响有多夸张呢？其用户隐私数据甚至被明码标价的在暗网中出售。其中就包括: 用户的ID，用户姓名，用户登录密码等隐私数据。这是一件多么可怕的事情呢？。对于普通人来说，可能用处不大。但对于商业公司，黑客来说。那就是巨大的商业价值。殊不知对于用户来说那就是巨大危害。怎么说呢？商业公司就可以拿这些数据用来作营销，做推广。甚至又可以被出卖。对于不存好心的黑客来说，其危害更加不可想象！

真的还有隐私吗？

  在生活中，不知道你有没有遇到以下几种场景：

  1. 陌生来电可以准确的说出自己的个人信息！

  2. 浏览某内容后，所有APP都会推荐相关内容！

  3. 租房，购车，考试后会持续遭到持续骚扰！

  4. 注册金融类服务后，会持续遭遇贷款保险推销！

据《中国个人信息与隐私保护报告》中显示，陌生来电可以准确的说出自己的个人信息比例高达81%。甚至有人一天可以接到高大5-6起。 更可怕的是。用户都不知道是谁泄漏了自己的信息。

是谁泄漏了你的隐私？

  随着互联网时代的进步，特别是移动互联网的快速发展。现在的企业，做的任何产品都是不管三七二十一。先拿到通讯录，通讯记录，短信记录等用户隐私权限再说。还美其名曰的说获取隐私数据是为了更好的用户体验。(特么的别骗自己了，又特么的想甩锅)。有些APP甚至流氓到，不给权限就不让安装。因此就有很多企业拥有着用户大量的隐私数据。其中就不乏：用户真实姓名，身份证号，手机号，身份证地址等等。对于安装时，称之为用户。获取到隐私数据后，后面的事情就真不关用户什么事情了。还记得前一段时间百度CEO李彦宏则称中国用户愿以隐私换便捷。被喷的不成样子。我倒觉得是一件好事情。因为这代表用户开始注重自己的隐私了。

日常中的隐私数据泄漏

  上面说的是企业将用户隐私数据的滥用。在日常中，其实也有很多隐私数据都是自己主动泄漏的！

  1. 外卖单上的联系人，联系人电话，住址。

  2. 快递单上的收件人姓名，手机号，收件地址等等。

  3. 火车票，汽车票，飞机票等行程单的。

这些都是隐私数据泄漏的途径。都要妥善处理！

最后

  对于用户而言。我们应该保护好自己的隐私数据。对于企业来说。更应该有责任保护好用户的隐私数据！

]]> 公司禁用QQ，破解方法（洋葱tor 安装配置） https://uzzz.org/article/1254.html Thu, 22 Apr 2010 08:54:45 +0000 http://wp.uzzz.org/article/1254.html

在公司还在烦恼为只能代理上网，有些网页还被禁，QQ上不了吗？那么哥今天将帮你解决这个烦恼，因为哥刚配置成功，效果不错，还很爽。

时间：2010-04-22

地点：上海某个角落

人物：我

原因：内部不能上QQ，其它都行，网速也快。每天手机上QQ有点纠结。

使用的工具：tor(vidalia-bundle)+火狐

我下载的是：（vidalia-bundle-0.2.1.19-0.1.15）+（Firefox/3.6.3）

 

Tor使用教程之Tor的安装与使用：http://article.wxiu.com/yyrj/network/200908/05-6243.html

 按照上面的去安装，是没什么问题，但我中间也出点小问题，就是安装torbutton版本跟火狐不兼容。既然不兼容，那我们就找个兼容的，在它弹出的插件搜索框搜索torbutton，找到后安装即可。

 

完成后有个洋葱一样的图标，自动的运行一个vidalia的面板，上面有个进度条老卡在中间就不动了（见图1），

需要设置，设置如下：

1、点击“设置”，选择“网络”，选上“我使用代理服务器连接到网络”设置公司上网的代理和端口，

将“我的isp阻挡对tor网络的连接”选中

然后将第一步获取到的网桥ip一个一个写进去。

2,设定好以后点击确定，然后停止tor，再启动，如果还是没有连上不要着急，多试几次，因为是特殊FQ软件，所以很不稳定，得根据网桥的速度来。如果连接成功，洋葱就会变绿。

3，试试上上网页，打开火狐，点击网页右下角，要显示绿色的“tor已启用”。网速有点慢，要等会儿。

4，如果网页能上了，再试试被禁掉的网页能不能打开？如：www.qq.com

5,如果被禁的网页都能上了，那QQ也能登陆。设置如下：
网络设置：协议“HTTP”，IP=127.0.0.1，端口=8118（其实这些是按照火狐里面网络设置自动设置的来的）

6，如果一切顺利的话，就能成功。

 

获取网桥地址的方法：

1、需要GMAIL邮箱

2、用GMIAL邮箱发送信息到一个用户，自动获取网桥地址

用Gmail发送邮件到bridges@torproject.org ，正文为“get bridges”，基本上同时收到如下相似回复：

 bridge 195.43.157.84:443
 bridge 204.74.213.53:443
 bridge 89.0.137.47:9001

]]> 塞翁失马 https://uzzz.org/article/2701.html Sat, 30 Sep 2006 07:16:00 +0000 https://uzzz.org/article/2701.html

 昨天，下班之后，一如既往地打开电脑，上网。突然显示器慢慢变暗，直到什么也看不到为止。

这种情况的出现已经不是第一次，因此我也没在意，关机，和往常一样，将插头拔下重插了一遍，开机，满怀信心地等待熟悉的开机画面出现…

嗯？还没出现，这下有点慌了，再重复了一遍刚才的动作，没反应。还不死心，将显示器接到别人的主机上试了一下，涛声依旧。

直到这时候，我才知道，大事不妙了。我的显示器坏了。

今年五一的时候，也是在放假前的一个晚上，一场天灾把我的主板年轻的生命给夺了去。不到半年的功夫，我可怜的显示器又步了它的后尘！

气极之下，反复的开机操作，希望有奇迹发生。果然发现操作系统也不能启动了，老天实在是太爱开玩笑了。

折腾到凌晨，终于通过系统还原功能，将电脑抢救了回来。

一天无精打彩，下午，也就是刚刚不久，同事告诉我有我的快递，打开一看，哈哈，终于盼到了久违的证件。也算对我不薄了，风雨过后就见彩虹。
