Tor Browser 8.5a8 发布了，此版本的主要变化是将 Firefox 更新到 60.5.1esr，修复了 Skia 库中的一些漏洞。

具体如下：

所有平台

  • Firefox 更新到 60.5.1esr

  • HTTPS Everywhere 更新到 2019.1.31

  • Bug 29378: 从默认网桥中删除 83.212.101.3

  • Bug 29349: 从 meek helper user.js 中删除 network.http.spdy.*

构建系统

  • 全平台

    • Bug 29235: 为 HTTPS Everywhere 构建自己的 python 3.6 版本

    • Bug 29167: Go 更新到 1.11.5

  • Linux

    • Bug 29183: linux-x86_64 上使用 linux-x86_64 语言包

官方表示，通常会确保每个 Tor 版本构建的可重现性，会在发布新版本之前让两个单独的人员进行构建，但是由于本周有些人无法访问，所对此 alpha 版本的验证有一些延迟。由于此版本包含希望快速发布的重要安全修复程序，因此官方将在之后检查构建的可重现性。

详情查看发布公告。
