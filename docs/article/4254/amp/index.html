3、结合其它软件使用方法

TOR Browser 内置了 TOR。这个内置的 TOR 会在本机地址（127.0.0.1）开启一个 SOCKS 端口，端口号是 9150

因此，如果某个网络软件跟 TOR Browser 安装在同一个操作系统中，并且该软件支持 SOCKS 代理，那么该软件就可以利用 TOR Browser 内置的 TOR进行工作。

例1：IE浏览 + TOR

假如你不喜欢 TOR Browser 内置的 Firefox，喜欢用自己的 IE。那么你可以设置 IE，让 IE走 TOR 提供的 SOCKS 代理。
