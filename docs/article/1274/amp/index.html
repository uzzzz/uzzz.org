一、tor的安装
1.添加tor的源
$sudo gedit /etc/apt/sources.list 
打开以后在最下面添加：

deb      http://mirror.noreply.org/pub/tor etch main
deb-src http://mirror.noreply.org/pub/tor etch main

$ sudo apt-get update

然后可能会有：

W: GPG签名验证错误： http://mirror.noreply.org etch Release: 由于没有公钥，下列签名无法进行验证： NO_PUBKEY CFF71CB3AFA44BDD

 

这时就要添加公钥

注意： NO_PUBKEY后面时数字时随机的，下一步需要数字的最后8位


2.添加公钥

$ gpg –keyserver subkeys.pgp.net –recv AFA44BDD
（AFA44BDD就是最后那8位数字）

$ gpg –export –armor AFA44BDD | sudo apt-key add –

应该提示OK了


3.更新源，安装tor

$ sudo apt-get update
这次应该正常完成了


4.下载需要的libevent包
从http://mirror.lupaworld.com/ubuntu/pool/main/libe/libevent/找自己版本下载安装

这里很奇怪我用安装libevent-1.4-2_1.4.13后不能用，只能再安装libevent1_1.3e-3版本……


$ sudo apt-get install tor privoxy
$ sudo apt-get install vidalia

vidalia是tor图形控制工具，windows下也有……

5.取得网桥

给 bridges@torproject.org
发邮件，题目和内容都为纯文本的 get bridges
无需等待多久，自动回复
在回信中可以获取网桥地址

在validia中，点”设定->网络”，选择我的ISP阻挡了对Tor的连接，然后在下面将邮件里面的数据一条条复制进去。只需IP:端口。
成功后validia图标会变成绿色（就是那个小洋葱）
