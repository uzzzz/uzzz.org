修改putty非常暗的注释字体颜色

fandyvon
6 years ago 

转载自http://m.blog.csdn.net/blog/astrotycoon/9729961 个人很喜欢使用putty，不但小巧，也很喜欢他的默认配色方案。唯一美中不足的就是它的默认注释颜色，很暗淡，大白天有时根本看不清。        上网查了查，终于知道可以通过vim的hi（highlight）命令改变。

    以下是vim不同关键字字体颜色修改方法
    修改字符串颜色  hi String ctermfg =darkred
    修改类型颜色 hi Type ctermfg =yellow
    修改数字颜色 hi Number ctermfg =darkblue
    修改常量颜色 hi Constant ctermfg =blue
    修改声明颜色 hi Statement ctermfg =darkyellow
    通过hi Comment ctermfg=blue后就清晰多啦！
